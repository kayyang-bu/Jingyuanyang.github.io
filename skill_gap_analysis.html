1 Skill Gap Analysis

1.1 1. Purpose

This section evaluates the gap between our team’s current technical skill levels and the skills most demanded in the 2024 job market. Building on our Exploratory Data Analysis (EDA), we focus on key skill categories in data science, business analytics, and machine learning roles. The goal is to identify strengths, highlight gaps, and provide actionable recommendations for career development.

1.2 2. Market Skill Demand from EDA

Based on our EDA of the Lightcast dataset, job postings in data and analytics roles emphasize the following skill groups:

  • Programming: Python, R, Scala
  • Core Data Skills: SQL, statistics, data analysis, data visualization
  • Traditional BI Tools: Tableau, Power BI, Excel
  • Cloud Platforms: AWS, Azure, Databricks, Snowflake
  • AI/ML Tools: TensorFlow, PyTorch, scikit-learn, MLflow, SageMaker

These findings indicate that employers increasingly expect candidates to combine strong analytical foundations with technical and cloud-based capabilities.

1.4 4. Team Skill Visualization

1.4.1 Interpretation

The heatmap shows that the team is relatively strong in foundational skills such as Excel, Python, and data visualization. However, advanced skills such as cloud platforms, machine learning, and AI tools are relatively weak across most members.

This suggests that the team is well-prepared for basic data analysis tasks but may face challenges when applying advanced analytics techniques in real-world business environments.

1.6.1 Interpretation

The largest gaps appear in cloud platforms, machine learning, and AI-related tools. The team is relatively closer to market expectations in Excel, Python, and basic data analysis skills.

This indicates that while the team has a solid analytical foundation, there is a need to strengthen more advanced technical capabilities to remain competitive in the job market.

1.7 7. Recommendations for Skill Development

To address the identified gaps, the following actions are recommended:

  • Strengthen Python and SQL proficiency across all team members
  • Learn cloud platforms such as AWS or Azure
  • Develop machine learning skills using libraries like scikit-learn
  • Improve data visualization skills using Tableau, Power BI, or Plotly
  • Work on real-world projects to integrate analysis, modelling, and visualization

1.8 8. Conclusion

Overall, the team demonstrates a strong foundation in core analytics skills. However, gaps in advanced technical areas such as cloud computing and machine learning suggest the need for continued skill development.

By aligning learning efforts with market demand, the team can significantly improve its competitiveness in business analytics and data science roles.
